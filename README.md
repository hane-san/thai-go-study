# 旅行タイ語 街歩きノート

バンコクの屋台・市場・カフェ・移動で使う短いタイ語を、タイ文字・読み方・日本語・音声で確認できるスマホ向けHTML教材です。

## 使い方

- `index.html` をブラウザで開きます。
- 中央がレッスン、左スワイプで文字・数字、右スワイプで単語ストックです。
- タイ語カードをタップすると読み上げます。
- 下部のタイ数字タブで課を切り替えます。

## GitHub Pages

このフォルダの中身をリポジトリ直下に置き、GitHub Pages を有効化すれば公開できます。

## Design note

チャオプラヤー川の夕焼け、寺院のシルエット、バンコクのポップな屋台街をイメージした配色です。本文カードは視認性を優先して、白〜クリームの背景にしています。
