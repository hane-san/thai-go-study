# 旅行タイ語 街歩きノート

バンコクの屋台・市場・カフェ・寺院まわりで使いやすい旅行タイ語のスマホ向けノートです。

## 使い方

- `index.html` を GitHub Pages で公開します。
- 左スワイプで文字・数字、中央でレッスン、右スワイプで単語ストックを見られます。
- タイ語の文をタップすると、端末の Web Speech API で読み上げます。

## 公開ファイル

- `index.html` — アプリ本体
- `.nojekyll` — GitHub Pages が余計なJekyll処理をしないようにする空ファイル

## 更新メモ

- 音声速度を学習向けの落ち着いた速度に戻しました。
- Thai TTS は、端末・ブラウザに存在する場合、男性名らしいタイ語音声を優先します。
- 文法説明欄をスマホで読みやすい文字サイズと行間に調整しました。
